# Basic skeleton project for:

- SvelteKit
- Supabase 
- TailwindCSS
- DaisyUI

## Feel free to use

To use download/clone repositiory then download all packages using
```
npm install
```
Then to run the project run
```
npm run dev
```

If you have any questions leave a issue or something ¯\_(ツ)_/¯
